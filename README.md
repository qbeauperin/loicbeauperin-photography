# Loic Beauperin - Photography

Powered with [11ty](https://github.com/11ty/eleventy/) and [Static CMS](https://github.com/StaticJsCMS/static-cms).

## Setup

```
npm install
```

## Run locally

```
npm start
```

## Access the admin

Go to `/admin` to access the Static CMS interface.

## Build

```
npx @11ty/eleventy
```
